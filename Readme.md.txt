# Compliance Mapping Web App

## Overview
This web application helps map and explore multiple compliance frameworks:
- NIST 800-53
- HIPAA
- ISO 27001
- PCI-DSS
- SOC 2

## Features
- Interactive UI
- Framework comparison
- Control mapping
- Easy navigation

## Tech Stack
- HTML, CSS, JavaScript
